# 프로젝트 소개
자취생 공동구매 앱의 admin page.
자취생 공동구매 앱 프로젝트 이름이 나누리임.

앱 프론트 IOS개발자 2명(기획 1명),
서버 Django 개발자 1명,
(나:)웹 프론트 React 개발자(기획)과
더불어 개발중

# 기술스택
- React (no Redux)
- TypeScript
- MaterialUI
- formik
- Context
- KakaoLogin
